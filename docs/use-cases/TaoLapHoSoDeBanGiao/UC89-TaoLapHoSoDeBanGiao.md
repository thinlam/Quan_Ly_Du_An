# Use Case 89: Tạo lập hồ sơ để bàn giao

<!-- region:nội dung-gốc -->
## Thông tin gốc

**Tên tác nhân:** Phòng được giao chủ trì

**Mô tả:**
- Cho phép xem danh sách hồ sơ đã tạo lập
- Cho phép tạo mới hồ sơ để bàn giao
- Cho phép đưa (chọn từng file hoặc tất cả) file (VB, QĐ, HĐ, …) đã được đính kèm trong quá trình thực hiện dự án để đưa vào hồ sơ để bàn giao
- Cho phép đính kèm thêm các file hồ sơ từ máy tính để thêm vào hồ sơ để bàn giao
- Cho phép chỉnh sửa, hoàn chỉnh hồ sơ tạo lập
- Cho phép kết xuất excel danh sách VB, QĐ, HĐ, … từ hồ sơ tạo lập
